# Loreto, territorio posible

Presentación web para GitHub Pages del Trabajo Final 2026 del Doctorado en Desarrollo Territorial (UNRC).

## Archivos

- `index.html`: presentación principal.
- `.nojekyll`: configuración para GitHub Pages.
- `img/`: imágenes, mapas y láminas documentales usadas en la presentación.

## Publicación en GitHub Pages

1. Subir `index.html`, `.nojekyll` y la carpeta `img/` a la raíz del repositorio.
2. Activar: `Settings → Pages → Deploy from branch → main → /root`.
3. Abrir la URL de GitHub Pages.

## Navegación

- Flecha derecha: siguiente placa.
- Flecha izquierda: placa anterior.
- También funcionan los botones inferiores.
